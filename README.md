# squid-proxy
This is an automatic script to install Squid-proxy with basic authentication for Linux Ubuntu.
## Installation
1. Download or clone this repo.
    ```bash
    git clone [this repo]
        cd LemonDCScript
    ```
2. Make the file `lemon.sh` is executable.
    ```bash
    chmod +x lemon.sh
    ```
3. Execute `lemon.sh`
    ```bash
    ./lemon.sh
    ```
    ### Enjoy !
